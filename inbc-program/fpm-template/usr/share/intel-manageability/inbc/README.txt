## Running INBC command-line utility tool

To run INBC as a binary:

1. To trigger firmware update : sudo inbc --trigger firmware-update --path /var/cache/manageability/repository-tool/fip-hddl2.bin --target None --vendor Intel --manufacturer intel --product kmb-hddl2 --release_date 2020-08-25

2. To trigger system update : sudo inbc --trigger system-update --path /var/cache/manageability/core-image-minimal-20200824131836.dm-verity.mender --target None
